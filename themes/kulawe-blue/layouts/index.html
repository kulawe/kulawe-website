Kulawe Limited

Kulawe Limited is a consultancy and training company. We specialise in the Scrum project management framework and the ‘git’ version control tool.

We use the Scrum project management framework to help software development teams deliver better software faster. We work with organisations to help them introduce Scrum or improve their use of Scrum.

git is a modern version control solution. We offer git training courses to developers and managers. We help software development teams transition from their existing version control solutions to git. We also help them integrate git into their existing systems to provide an application lifecycle management solution.
